Feature: Create Hobby Project

User Story: 
	As a user, I want to be able to create hobby,
	So that I can set targets/goals and track my progress over time in it.

Scenario1: 
	User wants to create a new hobby so they can put their goals in it
        Given that User is already logged in the user should see all the features
        The user should be able to create their own hobby Project

Test Design: 
        Log in as User and on the "Create Hobby" page
        Click on "Create Hobby button"
        Fill in the "Title" field with "Learn Guitar"
        Select the Wallpaper you want to set for your hobby 
        Click on the "Create Hobby" button

Outcome:
       User should see The new hobby appear on "Hobby" page with the correct details
